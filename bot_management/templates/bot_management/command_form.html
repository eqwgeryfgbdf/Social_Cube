Command Help
Creating Discord Commands

Discord slash commands allow users to interact with your bot using the / prefix in Discord.

Command Types
  • CHAT_INPUT - Standard slash command with parameters
  • USER - Command that appears in user context menus
  • MESSAGE - Command that appears in message context menus
Naming Rules
  • Names must be between 1-32 characters
  • Must match ^[-_\p{L}\p{N}\p{sc=Deva}\p{sc=Thai}]{1,32}$
  • Lowercase for CHAT_INPUT commands
Permissions

Default member permissions define which users can use the command based on their Discord permissions.

Enter a number representing the sum of all permission flags. Learn more

Guild vs Global

Guild commands are specific to a single server and update instantly.

Global commands work in all servers but take up to an hour to update.

Tip

After creating a command, add options to make it interactive and collect user input.
